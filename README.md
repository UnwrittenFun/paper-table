# paper-table

A material design table for polymer

## Install
```
bower install UnwrittenFun/paper-table --save
```

## Usage
```html
<paper-table>
  <div heading>
    <span title>Your title</span>
    <span flex></span>
    <paper-icon-button icon="filter-list"></paper-icon-button>
    <paper-icon-button icon="more-vert"></paper-icon-button>
  </div>
  <div thead>
    <paper-td>..</paper-td>
    <paper-td>..</paper-td>
    <paper-td>..</paper-td>
  </div>
  <div body>
    <paper-tr>
      <paper-td>..</paper-td>
      <paper-td>..</paper-td>
      <paper-td>..</paper-td>
    </paper-tr>
  </div>
</paper-table>
```
